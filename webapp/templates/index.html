Win Probability Framework

Decisions are evaluated using win probability which naturally accounts for game context. A run in the 9th inning of a tied game is worth more than a run in the 5th inning of a blowout.

WP_send = P(score) × WP(scored) + P(out) × WP(out)
Expected win probability if the runner is sent, weighted by the chance they score vs. get thrown out.

WP_hold = WP(runner at 3B)
Win probability with the runner staying at third base after the single.

Decision: Send if WP_send > WP_hold
